2. Layout (Diseño)

Fixed (Fijo)

Ancho fijo: 600px
No cambia con el tamaño de la ventana
width: 600px;

Elastic (Elástico)

Ancho: 80%
Se adapta proporcionalmente al contenedor
width: 80%;

Fluid (Fluido)

Ancho: 100%
Ocupa todo el espacio disponible
width: 100%;

Hybrid (Híbrido)

max-width: 700px + width: 90%
Combina lo mejor: fluido hasta un límite máximo
max-width: 700px;
width: 90%;

3. Box-Sizing: Border-Box

Con box-sizing: border-box
El width incluye padding y border. ¡Mucho más fácil de calcular!

Este proyecto usa border-box para todos los elementos (*)

* {
  box-sizing: border-box;
}

6. Columnas CSS

Las columnas CSS permiten distribuir contenido automáticamente en múltiples columnas, similar a un periódico. Este texto se distribuye en 3 columnas con un espacio de 30px entre ellas.

La propiedad column-rule añade una línea divisoria entre columnas, haciendo la separación más visual y profesional. Es como tener un border entre columnas.

El contenido fluye naturalmente de una columna a otra, y es responsive por naturaleza. Si reduces el ancho de la ventana, las columnas se ajustan automáticamente.

column-count: 3;
column-gap: 30px;
column-rule: 3px solid #667eea;

7. Flotantes (Float)

Float: Left
Este elemento flota a la izquierda. El contenido fluye alrededor de él.
Float: Right
Este elemento flota a la derecha. Perfecto para layouts de dos columnas.
Clear: Both
Este elemento limpia los floats anteriores y aparece debajo de ambos.

🔧 Clearfix Hack aplicado: El contenedor usa ::after con clear: both para contener los elementos flotantes correctamente.

.float-left { float: left; }
.float-right { float: right; }
.clear-both { clear: both; }

/* Clearfix Hack */
.float-container::after {
  content: "";
  display: table;
  clear: both;
}
